Combining Networks:

 An older user interface that communicated with users through textual messages using a keyboard and monitor screen.
kernel The internal part of an operating system
device driver A component of the kernel which are the software units that communicate with the controllers (or at times, directly with the peripheral devices) to carry out operations on the peripheral devices attached to the machine.
file manager Coordinates the use of the machine's mass storage facilities.
memory manager Coordinates the machine's use of main memory.
virtual memory Space created by the memory manager to allow the computer to function as if it had additional memory.

Client/Server Model

 A procedure that transfers the operating system from mass storage (where it is permanently stored) into main memory.
ROM Read-only Memory is special nonvolatile memory that can be read but not altered
deadlock A resource allocation problem in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
spooling A technique of holding data for output at a later but more convenient time.
